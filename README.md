# Arcanum

Arcanum is a tower defense game developed during a GameJam for class "Introduction to Game Development" at Brno University of Technology (VUT FIT). Theme of the GameJam was *"Defense"*.


## Mechanics

### Use different towers
Game offers 5 mage towers to defend your cottage from enemies, each Mage specializes in different things, for example: an Ice Mage is good at slowing in an AoE, but offering no damage, while a Lightning Mage offers Long Range damage, with less damage .

### Upgrade your towers
Each mage tower can be upgraded using the in-game currency you gain by killing enemies. Each mage has a different upgradable stat.
